3D Spring Physics
=====================

Controls:
  w/a/s/d - movement
  Right click - camera
  Left click - depends on mode (selected by keyboard):
    0 - (nothing)
    1 - Add model
    2 - Remove model
    3 - Add light
    4 - Add spring
    5 - Add two-way spring
    6 - Remove spring
    7 - Toggle gravity
    8 - Grab
